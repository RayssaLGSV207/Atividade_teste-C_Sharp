# 📝 ConsoleApp1 - Hello World

## 📋 Descrição

Um programa simples em C# que exibe a mensagem "Hello, World!" no console. Este é um projeto básico para demonstrar a estrutura de um aplicativo console .NET.

## 🛠️ Tecnologias Utilizadas

- **.NET 10.0** - Framework de desenvolvimento
- **C#** - Linguagem de programação
- **Console Application** - Aplicação de linha de comando

## 📁 Estrutura do Projeto

```
ConsoleApp1/
├── Program.cs              # Código fonte principal
├── ConsoleApp1.csproj      # Arquivo de configuração do projeto
└── README.md               # Documentação do projeto
```

## 🚀 Como Executar

### Pré-requisitos

- [.NET 10.0 SDK](https://dotnet.microsoft.com/download) instalado

### Passos para execução

1. **Clone ou baixe o projeto**
   ```bash
   git clone <url-do-repositorio>
   cd ConsoleApp1
   ```

2. **Execute o programa**
   ```bash
   dotnet run
   ```

3. **Resultado esperado**
   ```
   Hello, World!
   ```

## 💻 Exemplo de Uso

```bash
# Navegar até a pasta do projeto
cd C:\caminho\para\ConsoleApp1

# Executar a aplicação
dotnet run

# Saída:
# Hello, World!
```

## 🔧 Comandos Úteis

| Comando | Descrição |
